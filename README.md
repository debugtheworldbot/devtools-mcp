# Chrome DevTools MCP Workshop

Google DevFest 2024 Workshop 活动网站

## 关于这个项目

这是一个记录 Chrome DevTools MCP Workshop 的活动网站，采用 Linear 风格设计。

### 页面

- **首页**: Workshop 总体信息
- **GDG & DevFest**: 介绍 Google Developer Groups 和 DevFest
- **DevTools MCP**: Chrome DevTools MCP Server 详细介绍
- **关于我**: 皮皮猪的个人介绍

### 技术栈

- 纯 HTML + CSS
- 响应式设计
- 现代渐变效果
- GitHub Pages 部署

## 本地运行

```bash
python3 -m http.server 8080
```

然后访问 http://localhost:8080

## 部署

本项目使用 GitHub Actions 自动部署到 GitHub Pages。
